# 📝 Group To-Do Task List Application

## 📌 Project Overview
This project is a web-based To-Do Task List application developed as a group assignment.  
The purpose of this project is to demonstrate teamwork, Git branching, and feature integration using GitHub.

Each group member implemented one new feature in their own Git branch and submitted a pull request to be merged into the main branch.

---

## 🎯 Objectives
- Practice collaborative development using Git and GitHub
- Understand branching and merging
- Build a functional web application using HTML, CSS, and JavaScript
- Implement new features independently

---

## 🛠 Technologies Used
- HTML5

- Git & GitHub

## 👨‍💻 Individual Contributions

Each group member worked on a separate branch:

| Feature | Branch Name |
|----------|-------------|
| Due Date | feature-due-date |
| Priority Level | feature-priority |
| Search Task | feature-search |
| Task Completion | feature-complete |
| Edit Task | feature-edit |

---

## 🔄 Git Workflow

1. Clone the repository
2. Create a new branch
3. Implement assigned feature
4. Commit changes
5. Push branch to GitHub
6. Submit a Pull Request
7. Merge into main branch

---

## ▶ How to Run the Project

1. Clone the repository:
   git clone <repository-link>

2. Open the project folder.

3. Open `index.html` in a web browser.
